04 System Architecture

Oracle permissions are restricted, calls are rate-limited, and address changes are timelocked to maintain trust-minimized operation.

4.1 Hybrid Oracle Design

SCANDAL uses a hybrid oracle architecture, which is standard across production Web3 systems.

Off-chain layer

  • Collects public news data
  • Performs AI-assisted sentiment analysis
  • Produces a numerical sentiment score

On-chain layer

  • Validates oracle calls
  • Enforces supply changes
  • Settles prediction rounds
  • Distributes fees and rewards

The Oracle:

  • does not hold user funds
  • does not control balances
  • does not execute economic actions

All critical logic is enforced on-chain.

4.2 Oracle Security Model

  • Oracle permissions are restricted
  • Oracle calls are rate-limited
  • Oracle address changes are timelocked
  • Users can exit before changes take effect

This design minimizes trust while maintaining real-time responsiveness.
